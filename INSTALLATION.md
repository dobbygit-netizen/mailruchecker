# Инструкция по установке Mail.ru Checker

## Установка расширения

### Способ 1: Установка из исходного кода (Рекомендуется)

1. **Скачайте расширение**
   - Скачайте ZIP-архив репозитория или клонируйте его
   - Распакуйте архив в удобное место на компьютере

2. **Откройте страницу расширений в браузере**
   - **Chrome**: Перейдите по адресу `chrome://extensions/`
   - **Edge**: Перейдите по адресу `edge://extensions/`
   - **Другие Chromium браузеры**: `browser://extensions/`

3. **Включите режим разработчика**
   - Найдите переключатель "Режим разработчика" в правом верхнем углу
   - Активируйте его

4. **Загрузите расширение**
   - Нажмите кнопку "Загрузить распакованное расширение"
   - Выберите папку с распакованным расширением
   - Расширение должно появиться в списке

5. **Закрепите расширение**
   - Нажмите на иконку пазла (расширения) в панели инструментов
   - Найдите "Mail.ru Checker" и нажмите на значок булавки
   - Теперь иконка расширения всегда будет видна

## Первоначальная настройка

### Шаг 1: Войдите в Mail.ru

Перед настройкой расширения убедитесь, что вы вошли в свои аккаунты Mail.ru:

1. Откройте новую вкладку
2. Перейдите на https://e.mail.ru
3. Войдите в свой аккаунт
4. Если у вас несколько аккаунтов, войдите в каждый из них в разных окнах браузера или профилях

### Шаг 2: Настройте расширение

1. **Откройте настройки**
   - Нажмите на иконку расширения в панели инструментов
   - Нажмите кнопку "⚙️ Настройки"

2. **Добавьте email адреса**
   - В разделе "Аккаунты Mail.ru" введите свой email адрес
   - Нажмите "+ Добавить аккаунт" для добавления дополнительных адресов
   - Формат: `example@mail.ru`

3. **Настройте интервал проверки**
   - По умолчанию: 5 минут
   - Диапазон: от 1 до 60 минут
   - Рекомендуется: 3-5 минут для оптимальной работы

4. **Настройте уведомления**
   - ✅ Показывать уведомления о новых письмах (рекомендуется)
   - ⬜ Звуковое уведомление (опционально)

5. **Сохраните настройки**
   - Нажмите "Сохранить настройки"
   - Дождитесь сообщения об успешном сохранении

### Шаг 3: Первая проверка

1. Закройте страницу настроек
2. Нажмите на иконку расширения
3. Нажмите кнопку "🔄 Проверить сейчас"
4. Дождитесь завершения проверки
5. Вы увидите количество непрочитанных писем

## Проверка работоспособности

### Что должно работать:

✅ **Иконка расширения**
- Видна в панели инструментов браузера
- Показывает красный счетчик при наличии непрочитанных писем

✅ **Всплывающее окно**
- Открывается при клике на иконку
- Отображает общее количество непрочитанных писем
- Показывает список всех настроенных аккаунтов
- Отображает время последней проверки

✅ **Уведомления**
- Приходят при появлении новых писем
- Содержат информацию о количестве писем
- Клик открывает почту

✅ **Автоматическая проверка**
- Выполняется согласно заданному интервалу
- Работает в фоновом режиме

## Решение проблем при установке

### Расширение не загружается

**Проблема**: Ошибка при загрузке расширения

**Решение**:
1. Убедитесь, что выбрана правильная папка с файлами расширения
2. Проверьте, что файл `manifest.json` находится в корне папки
3. Убедитесь, что все файлы распакованы правильно
4. Попробуйте перезагрузить браузер

### Ошибка "Manifest file is missing or unreadable"

**Проблема**: Не найден файл манифеста

**Решение**:
1. Убедитесь, что файл `manifest.json` существует
2. Проверьте, что вы выбрали корневую папку проекта
3. Убедитесь, что файл не поврежден

### Расширение загружено, но не работает

**Проблема**: Расширение установлено, но не показывает данные

**Решение**:
1. Войдите в Mail.ru в браузере
2. Добавьте email адреса в настройках расширения
3. Нажмите "Проверить сейчас" для ручной проверки
4. Проверьте консоль разработчика (F12) на наличие ошибок

### Нет разрешений для уведомлений

**Проблема**: Уведомления не приходят

**Решение**:
1. Откройте настройки браузера
2. Перейдите в раздел "Конфиденциальность и безопасность"
3. Найдите раздел "Уведомления"
4. Убедитесь, что уведомления разрешены для расширений

## Обновление расширения

1. Скачайте новую версию расширения
2. Удалите старую версию на странице `chrome://extensions/`
3. Загрузите новую версию согласно инструкции выше
4. Настройки обычно сохраняются автоматически

## Удаление расширения

1. Перейдите на страницу `chrome://extensions/`
2. Найдите "Mail.ru Checker"
3. Нажмите "Удалить"
4. Подтвердите удаление

Все локальные данные (настройки, список аккаунтов) будут удалены вместе с расширением.

## Поддержка нескольких аккаунтов

Расширение поддерживает проверку нескольких аккаунтов Mail.ru одновременно:

1. Войдите во все аккаунты в браузере
2. Добавьте все email адреса в настройках расширения
3. Расширение будет проверять каждый аккаунт отдельно
4. В popup вы увидите статус каждого аккаунта

## Советы по использованию

💡 **Оптимальный интервал**: 3-5 минут обеспечивают баланс между актуальностью данных и нагрузкой

💡 **Множественные аккаунты**: Используйте разные профили браузера для лучшей изоляции аккаунтов

💡 **Конфиденциальность**: Расширение хранит только email адреса локально, пароли не требуются

💡 **Производительность**: Расширение работает в фоновом режиме и почти не потребляет ресурсы
