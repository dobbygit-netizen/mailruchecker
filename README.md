# Mail.ru Checker

Расширение для Chrome/Edge для автоматической проверки почты Mail.ru с отображением количества непрочитанных писем.

## Функциональность

- ✅ Автоматическая проверка почты Mail.ru
- ✅ Отображение счетчика непрочитанных писем на иконке расширения
- ✅ Поддержка нескольких аккаунтов
- ✅ Настраиваемый интервал проверки (от 1 до 60 минут)
- ✅ Уведомления о новых письмах
- ✅ Быстрый доступ к почте из popup
- ✅ Детальная информация по каждому аккаунту

## Установка

### Из исходного кода

1. Скачайте или клонируйте этот репозиторий
2. Откройте Chrome/Edge и перейдите в `chrome://extensions/` (или `edge://extensions/`)
3. Включите "Режим разработчика" в правом верхнем углу
4. Нажмите "Загрузить распакованное расширение"
5. Выберите папку с расширением

## Настройка

1. Нажмите на иконку расширения в панели инструментов
2. Нажмите кнопку "⚙️ Настройки"
3. Добавьте свои email адреса Mail.ru
4. Настройте интервал проверки
5. Выберите нужные параметры уведомлений
6. Нажмите "Сохранить настройки"

## Использование

### Основные функции

- **Автоматическая проверка**: Расширение автоматически проверяет почту согласно заданному интервалу
- **Счетчик на иконке**: Красная цифра показывает общее количество непрочитанных писем
- **Popup**: Нажмите на иконку для просмотра детальной информации по каждому аккаунту
- **Быстрая проверка**: Кнопка "Проверить сейчас" для немедленной проверки
- **Открытие почты**: Кнопка "Открыть почту" для быстрого доступа к Mail.ru

### Уведомления

При появлении новых писем расширение покажет уведомление с:
- Общим количеством непрочитанных писем
- Разбивкой по аккаунтам (если их несколько)

Клик по уведомлению откроет почту в новой вкладке.

## Технические детали

### Архитектура

- **background.js**: Service Worker для фоновой проверки почты
- **offscreen.js**: Offscreen документ для парсинга страниц Mail.ru
- **popup.js**: Скрипт всплывающего окна
- **settings.js**: Скрипт страницы настроек

### Методы проверки

Расширение использует несколько методов для получения количества непрочитанных писем:

1. **API Mail.ru**: Первичный метод через официальное API
2. **Парсинг iframe**: Резервный метод через загрузку страницы в iframe
3. **Fetch запрос**: Альтернативный метод для получения данных

### Хранение данных

Все данные хранятся локально в `chrome.storage.local`:
- Настройки пользователя
- Список аккаунтов
- Результаты последней проверки
- Время последней проверки

## Разрешения

Расширение запрашивает следующие разрешения:

- `storage`: Для сохранения настроек и результатов
- `notifications`: Для показа уведомлений о новых письмах
- `offscreen`: Для проверки почты через offscreen документ
- `host_permissions` для `e.mail.ru` и `account.mail.ru`: Для доступа к почте Mail.ru

## Приватность

- Расширение не собирает и не передает ваши данные
- Все email адреса хранятся только локально в вашем браузере
- Проверка почты происходит напрямую с серверами Mail.ru
- Исходный код полностью открыт для проверки

## Ограничения

- Для проверки почты требуется активная сессия в Mail.ru
- При выходе из аккаунта в браузере проверка работать не будет
- Необходимо быть залогиненным в Mail.ru для каждого проверяемого аккаунта
- CORS ограничения могут препятствовать некоторым методам проверки

## Устранение неполадок

### Расширение не показывает непрочитанные письма

1. Убедитесь, что вы залогинены в Mail.ru в текущем браузере
2. Проверьте, что добавлены правильные email адреса в настройках
3. Попробуйте нажать "Проверить сейчас" для ручной проверки
4. Проверьте консоль разработчика на наличие ошибок

### Уведомления не приходят

1. Убедитесь, что уведомления включены в настройках расширения
2. Проверьте настройки уведомлений браузера
3. Убедитесь, что браузер имеет разрешение показывать уведомления

### Расширение перестало работать после обновления браузера

1. Перезагрузите расширение на странице `chrome://extensions/`
2. При необходимости переустановите расширение
3. Проверьте совместимость с текущей версией браузера
